As a rule, follow the current standards when possible. If HTML5 and CSS3 are the current accepted standards, those are the conventions you should try to attain. Google has a well defined and maintained HTML and CSS guideline on GitHub here: http://google.github.io/styleguide/htmlcssguide.xml.

In addition to the Google HTML and CSS guidelines, Mark Otto has a an excellent reference here: http://codeguide.co/, that dives deeper into some specifics like how best to handle boolean operators in HTML and nesting SASS.
